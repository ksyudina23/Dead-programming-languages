Каталог языков программирования. Исторический раздел

Языки программирования, вышедшие из активного употребления, но оказавшие значительное влияние на развитие вычислительной техники. По каждому языку ниже приведены исторические сведения, технические характеристики и примеры кода.

COBOL
COmmon Business-Oriented Language
Год создания: 1959
Автор: Грейс Хоппер, комитет CODASYL
Классификация: Императивный, процедурный язык
Создан по заказу Министерства обороны США для обработки больших объёмов данных. Отличался близостью к английскому языку, что делало программы понятными для менеджеров и аудиторов. Доминировал в банковской сфере, страховании, государственных учреждениях до 2000-ых годов.
Вложил основы работы с записями и файлами. Повлиял на SQL, PL/I. Показал важность стандартизации языков.
Пример кода:
IDENTIFICATION DIVISION.
PROGRAM-ID. HELLO.
PROCEDURE DIVISION.
          DISPLAY "Hello, world!".
          STOP RUN.
Ada
Год создания: 1983
Автор: Жан Ишбиа
Классификация: Структурный, объектно-ориентированный, многозадачный язык
Разработан по заказу министерства обороны США как единый язык для всех военных проектов. Назван в честь Ады Лавлейс. Отличался исключительной надежностью, поддержкой многозадачности, строгим контролем типов. Использование в авионике (Boeing 777), железнодорожных системах (Парижское метро), европейском проекте Ariane.
Показал важность надежности и безопасности программирования. Ввел поддержку многозадачности на уровне языка. Повлиял на Java (интерфейсы, пакеты), Python, C#.
Пример кода:
with Ada.Text_IO;
procedure Hello is
begin
        Ada.Text_IO.Put_Line("Hello, World!");
end Hello;
Pascal
Год создания: 1970
Автор: Никлаус Вирт
Классификация: Структурный, императивный язык
Создан как язык для обучения структурному программированию. Отличался строгой типизацией, понятным синтаксисом, поддержкой пользовательских типов данных. В 1980-1990ых годах был основным языком для обучения в школах и вузах. На основе Pascal создан Delphi (Object Pascal).
Оказал огромное влияние на языки Ada, Modula-2, Oberon. Строгая типизация и структурный подход стали стандартом индустрии.
Пример кода:
program Hello;
begin
       writeln("Hello, World!");
end.
FORTRAN
FORmula TRANslator
Год создания: 1957
Автор: Джон Бэкус, IBM
Классификация: Императивный, процедурный язык высокого уровня
Первый в мире язык программирования высокого уровня. Разработан для упрощения написания программ на IBM 704. Позволял писать формулы в математической нотации, что значительно ускорило разработку научных приложений. Активно использовался до 1990-х годов в физике, метеорологии, аэрокосмической отрасли.
Показал важность надежности и безопасности программирования. Ввел поддержку многозадачности на уровне языка. Повлиял на Java (интерфейсы, пакеты), Python, C#.
Пример кода:
PROGRAM HELLO
PRINT *, "Hello, World!"
END
